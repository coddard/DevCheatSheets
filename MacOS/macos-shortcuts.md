# 🖥️ macOS Keyboard Shortcuts  

**Supercharge your Mac workflow with emoji-powered shortcuts! 🚀✨**

## ⌨️ Modifier Keys

| Emoji | Key  | Description    |
| ----- | ---- | -------------- |
| 💪    | `⌘`  | Command (Cmd)  |
| 🎮    | `⌃`  | Control (Ctrl) |
| 🔓    | `⌥`  | Option (Alt)   |
| ⚙️    | `⇧`  | Shift          |
| 🔠    | `⇪`  | Caps Lock      |
| 🎯    | `Fn` | Function Key   |
